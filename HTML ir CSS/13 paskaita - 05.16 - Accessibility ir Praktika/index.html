Units

Units - matavimo vienetai, skirti nustatyti elementų dydžius: plotis, aukštis, teksto dydis, tarpai.

  • px
  • % - pagal tėvinio elemento dydį
  • vh - viewport height - matomos svetainės dalies aukštis (100 - visas aukštis)
  • vw - viewport width - matomos svetainės dalies plotis (100 - visas plotis)
  • rem - root em(emphemeral) - 16px - šakninis nustatymas
  • em - 16px - tėvinio elemento nustatymas
