Git ir Github

Git - programa, kuri versijuoja kodą.

Github - svetainė, kurioje keliamas (versijuotas) kodas.

Git setup

  1. Įsirašyti Git programą
  2. Pasitikrinti ar tikrai įrašytas Git'as. Terminale panaudoti komandą "git -v".
  3. Susikurti Github paskyrą.
  4. Konfigūruoti git'ą globaliai (jūsų naudojamame įrenginyje)
    • git config --global user.name "GithubUserName"
    • git config --global user.email "GithubEmail"
    • git config --global core.editor "code"
    • git config --global core.autocrlf true|input (true jeigu windows | input jeigu mac|linux)
  5. Pasitikrinti ar teisingai atlikta konfigūracija. Terminale panaudoti komandą "git config --global --edit"
