Breakpoints

Breakpoint - ekrano lūžis - kai iš vieno išdėstymo, pavyzdžiui telefoninio dydžio, yra pakeičiamas elementų išdėstymas į kitą išdėstyma, pavyzdžiui plančetiniam dydžiui.
Breakpoint'ų galima prikurti kiek tik nori (reikėtų nepersitengti, dažniausiai daromi 3) ir ties kokiais nori dydžiais (yra šokie tokie standartai).

  1. x-small - portrait phones - 0px => 575px
  2. small - landscape phones - 576px => 767px
  3. medium - tablets - 768px => 991px
  4. large - laptop - 992px => 1199px
  5. x-large - desktop - 1200 => 1399px
  6. xx-large - large desktops -> 1400px => infinity

Tam, kad būtų sukurtas vadinamasis breakpoint'as CSS'e, reikia aprašyti jo media query.
Media query apsirašo šitaip: @media (min-width: number_px){ ... css ... }

Media query nesuteikia papildomo stilių prioriteto, dėl to media query reikia išrašyti teisingu eiliškumu.
Media query stiliai nepradingsta, pasiekus naują media query.
