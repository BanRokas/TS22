CSS - Cascading Style Sheets yra skirtas stilizuoti HTML'o elementus.

CSS rašymo būdai

  • Inline - Linijinis - CSS rašomas tag'o viduje, style atribute.
  • Internal - Vidinis - CSS rašomas HTML faile, style tag'o viduje.
  • External - Išorinis - CSS rašomas atskirame CSS faile, kuris yra prijungiamas prie HTML'o.

CSS Selektoriai

Developer Mozilla
W3 School

CSS selektoriai yra naudojami tam, kad CSS faile nurodytume, kokiems elementams norime suteikti stilius.

  • tag - span/p/h1... - yra parenkami visi HTML esantys elementai turintys nurodytą tag'ą.
  • * - yra parenkami visiškai visi elementai.
  • class - yra parenkami visi elementai turintys nurodytą klasę.
  • id - yra parenkas elementas turintis nurodytą ID.

Class

Vieną klasę galima priskirti daugeliui elementų. Vienas elementas gali turėti daugelį klasių.
CSS klasė aprašoma už taško simbolio "." susigalvotu (siūlomai) logišku pavadinimu. HTML'e klasė priskiriame suteikiant atributą "class" ir už lygybės kabutėse išrašomos klasė/s per tarpelį.

ID

Vienas ID gali būti priskirtas TIK vienam elementui. Vienas elementas gali turėti TIK vieną ID.
CSS ID aprašomas už grotelių simbolio "#" susigalvotu (siūlomai) logišku pavadinimu. HTML'e id priskiriame suteikiant atributą "id" ir už lygybės kabutėse parašomas id pavadinimas.

Spalvos

Spalvos gali būti užrašytos įvairiais būdais. Populiariausi spalvų aprašymo būdai yra: spalvos vardas, rbg kodas, hex kodas.

Spalvos vardas

Spalva gali būti suteikiama nurodant jos vardą. Vardinių spalvų yra ~140.

RGB kodas

RGB kodas susideda iš RedGreenBlue spalvų, kurioms galima skirti nuo 0 iki 255 kiekio. To pasekoje galima išgauti 16.777m spalvų.
RBG gali turėti permatomumą (alpha), tuomet kodas vadinamas RGBA ir turi nebe 3 reikšmes, bet 4, o tą 4'toji reikšmė yra nuo 0 iki 1, 0.001 žingsniu.

Hex kodas

Hex yra trumpinys hexadecimal žodžiui, kuris reiškia šešioliktainis. Kitaip pasakius hex kodas yra šešioliktainė skaičiavimo sistema.
Hex kodas susideda iš #RRGGBB, kiekviena raidė yra skaičius nuo 0 iki 15 parašytas šešioliktainėje sistemoje.
Pvz.: dešimtainėje 50 yra lygus hexadecimal 32.
Hex kaip ir RBG gali turėti spalvos permatomumą. Jis aprašomas kaip du extra hexadecimal simboliai gale. (nebe #RRGGBB, bet #RRGGBBAA)
Hex yra vienintelis spalvos aprašymo būdas, kuris gali būti trumpinamas, jeigu vienos spalvos hexdecimal simboliai yra vienodi. Bet jeigu jau trumpinamas kodas, turi būti trumpinamos visos spalvos.

Inline

Linijiniai elementai yra tokie elementai, kurie užima tik tiek vietos, kiek reikia jų viduje esančiam turiniui. Todėl jie gali atsistoti vienas šalia kito toje pačioje eilutėje.
Linijiniams elementams negalime suteikti pločio arba aukščio nustatymų.

Linijinis Linijinis

Block

Blokiniai elementai yra tokie elementai, kurie užima visą eilutę, nesvarbu ar jų turiniui tiek vietos reikia. Todėl jie negali atsistoti vienas šalia kito, netgi su pločio nustatymo pagalba.
Blokiniams elementams galime suteikti pločio arba aukščio nustatymus.
