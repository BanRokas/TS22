CSS Įvadas

CSS - Cascading Style Sheets yra skirtas stilizuoti HTML'o elementus.

CSS rašymo būdai

  • Inline - Linijinis - CSS rašomas tag'o viduje, style atribute.
  • Internal - Vidinis - CSS rašomas HTML faile, style tag'o viduje.
  • External - Išorinis - CSS rašomas atskirame CSS faile, kuris yra prijungiamas prie HTML'o.

Inline

Naudojamas norint pasitestuoti kaip atrodytų stilius pritaikytas tam elementui. Nepaliekamas galutiniame kode.

Pavyzdys Inline

Internal

Naudojamas norint pasitestuoti CSS selektorius ir stilius ARBA turint labai mažą ir unikalų HTML failą, kuriam nenori kurti papildomo CSS'o.

External

Dažniausiai naudojamas CSS rašymo būdas. Kai CSS rašomas atskirame faile, kuris yra prijungiamas prie HTML'o su link tag'o pagalba.
