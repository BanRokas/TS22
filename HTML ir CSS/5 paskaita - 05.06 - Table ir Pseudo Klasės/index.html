Table

  • table - lentelė
  • thead - atvaizduoja lentelės stulpelių pavadinimus
  • tbody - atvaizduoja lentelės duomenis
  • tfoot - atvaizduoja lentelės išvestines (suma/vidurkis/...)
  • tr - table row - sukuria lentelės eilutę
  • th - table heading - lentelės stulpelio pavadinimas
  • td - table data - lentelės duomens laukelis
  • colgroup - stulpelių grupė
  • col - stulpelių paskirstymas
  • caption - lentelės pavadinimas

Attribute Selectors

  • [attribute] - atributo vardas
  • [attribute="reiksme"] - atributo vardas ir tiksli reikšmė
  • [attribute~="reiksme"] - atributo vardas ir reikšmės žodis
  • [attribute*="reiksme"] - atributo vardas ir reikšmės dalis
  • [attribute^="reiksme"] - atributo vardas ir reiksme prasidedanti nurodyta seka
  • [attribute$="reiksme"] - atributo vardas ir reiksme pasibaigianti nurodyta seka
  • [attribute|="reiksme"] - atributo vardas ir reikšmė arba reikšmė su "-"

Pseudo Class

  • Nuorodų
    1. :link
    2. :visited
    3. :hover
    4. :active
  • :first-child - pirmasis vaikinis elementas
  • :last-child - paskutinis vaikinis elementas
  • :only-child - vienintelis vaikinis elementas
  • :first-of-type - pirmasis savo tipo elementas
  • :last-of-type - paskutinis savo tipo elementas
  • :only-of-type - vienintelis savo tipo elementas
  • :nth-child(n)
  • :nth-last-child(n)
  • :nth-of-type(n)
  • :nth-last-of-type(n)
  • Nebūtini
    • :root - kurti kintamuosius CSS'e (ir kitokiems dar dalykams)
    • :not(selector) - parenka viską, bet ne nurodytą selektorių
    • :empty - tag'as be turinio
    • :fullscreen - pajungus fullscreen mode'ą
    • :lang(language) - kalbos atributo selektorius
    • :target - kreipiamasi į elementą, į kurį nusinaviguoji su nuoroda

Icons

Norint naudotis ikonomis, reikia jas susi'import'uoti iš vieno ar kito puslapio.
Ikonos neturi savo tag'o, jos naudoja arba 'i' arba 'span' tag'us.
Tam, kad 'i' arba 'span' tag'ui suteikti ikoną, reikia pridėti klases.
Norint stilizuoti ikonas, reikia jas stilizuoti kaip tekstą, o ne kaip bloką.
