  • table - lentelė
  • thead - atvaizduoja lentelės stulpelių pavadinimus
  • tbody - atvaizduoja lentelės duomenis
  • tfoot - atvaizduoja lentelės išvestines (suma/vidurkis/...)
  • tr - table row - sukuria lentelės eilutę
  • th - table heading - lentelės stulpelio pavadinimas
  • td - table data - lentelės duomens laukelis
  • colgroup - stulpelių grupė
  • col - stulpelių paskirstymas
  • caption - lentelės pavadinimas
