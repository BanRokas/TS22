Form

Formos tag'ai

  • form - formos tag'as
  • label - formos įvesties etiketė/pavadinimas
  • input - formos įvesties laukas (įvairus)
  • textarea - formos įvesties laukas (dideliam kiekiui teksto)
  • select - opcijų pasirinkimo laukas (nedideliam kiekiui opcijų)
  • datalist - opcijų pasirinkimo laukas (dideliam kiekiui opcijų)
  • option - opcija
  • output - formos atsakymui atvaizduoti

Formos tag'ų atributai

  • type - nurodo input'o tipą
    • text - tekstas
    • password - slapyvardis (atvaizduoja kaip taškelius)
    • number - skaičius
    • email - elektroninis paštas
    • url - nuoroda
    • checkbox - daugelio pasirinkimas
    • radio - vieno pasirinkimas
    • submit - formos pateikimas
    • file - kompiuteryje esančio failo pasirinkimas
    • tel - telefonas
    • range - skaičiaus pasirinkimas slenkančioje juostoje
    • date - datos pasirinkimas (metų, mėnesio, dienos)
    • datetime-local - datos ir laiko pasirinkimas (metų, mėnesio, dienos, valandos, minutės)
    • month - mėnesio pasirinkimas
    • week - savaitės pasirinkimas
    • time - laiko pasirinkimas
    • color - spalva
    • button - mygtukas be funkcionalumo
    • image - formos pateikimas kaip paveiksliukas
    • hidden - paslėpta įvestis
    • reset - išvalyti formą
    • search - paieškos laukas
  • for - nurodo kuriai įvesčiai priskirti label'į
  • id - label sujungimui su įvestimi
  • name - suteikia įvesčiai pavadinimą
  • value - įvesties reikšmė
  • required - privalomas įvesties laukas
  • placeholder - pavyzdinis tekstas
  • min - minimali reikšmė (skaičiaus)
  • max - maksimali reikšmė (skaičiaus)
  • step - žingsnis (skaičiaus)
  • accept - kokio tipo failas priimtinas
  • pattern - nurodo kokia turi būti įvesties seka (regex dažniausiai)
  • rows - textarea eilučių kiekį (vizualaus dydžio pavidalas)
  • cols - textarea stulpelių kiekį (vizualaus dydžio pavidalas)
  • checked - radio arba checkbox išankstinis parinkimas
  • selected - select išankstinis parinkimas
  • disabled - išjungtas įvesties laukas
  • readonly - neleidžia keisti reikšmių
  • multiple - leidžia pasirinkti daugiau select'o opcijų
  • ...

Pseudo Class

  • :checked - pažymėtas (checkbox / radio)
  • :default - pradinis (su checked atributu)
  • :disabled - turintis disabled atributą
  • :enabled - neturintis disabled atributo
  • :focus - fokusuota įvestis
  • :in-range - įvestis ribose
  • :out-of-range - įvestis už ribų
  • :indeterminate - neapibrėžta reikšmė
  • :valid - tinkama įvesties reikšmė
  • :invalid - netinkama įvesties reikšmė
  • :optional - įvestis neturinti required atributo
  • :required - įvestis turinti required atributą
  • :readonly - įvestis turinti readonly atributą
  • :read-write - įvestis neturinti readonly atributo
  • ...

Pseudo Elements

  • ::placeholder - parenka placeholder'į
  • ::before - sukuria turintį už
  • ::after - sukuria turinį prieš
  • ::marker - sąrašo elementų žymės
  • ::selection - pažymėtas turinys
  • ::first-letter - parenka pirmąją raidę
  • ::first-line - parenką pirmąją eilutę
  • ::-webkit-scrollbar
  • ::-webkit-scrollbar-track
  • ::-webkit-scrollbar-thumb
  • ...
