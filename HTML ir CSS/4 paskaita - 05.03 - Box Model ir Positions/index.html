Box Model

  1. Content - turinys, centrinė dalis
  2. Padding - pamušalas, tarpas tarp turinio ir rėmelio
  3. Border - rėmelis, esantis tarp padding ir margin
  4. Margin - paraštė, esanti tarp rėmelio ir kitų elementų
  5. Position - modifikuota pozicija, neįtakojant kitų elementų pozicijos

Turinio dalį įtakoja turinio kiekis ir jo nustatymai, kaip width; height; font-size; font...

Padding

Padding yra tarpas tarp turinio ir rėmelio.
Padding gali būti nustatomas 8 būdais.

  • padding-top: px;
  • padding-right: px;
  • padding-bottom: px;
  • padding-left: px;
  • padding: top right bottom left;
  • padding: top right&left bottom;
  • padding: top&bottom right&left;
  • padding: all;

Border

Border - rėmelis yra juosta atskirianti elemento padding'ą nuo jo margin'o. Už border'io baigiasi elemento vidinė dalis (baigiasi fono spalva/paveiksliukas).

Border'iui galima suteikti spalvos, linijos storio, linijos tipo, kampų apvalinimo stilius. Skirtingus stilius galima suteikti skirtingoms kraštinėms/kampams.

  • border: width style color;
  • border-width: px;
  • border-style: solid/dashed/dotted/double...
  • border-color: color;
  • border-[kraštinė]: width style color;
  • border-[kraštinė]-color/style/width: ???;
  • border-radius: px;
  • border-radius: px px;

Margin

Margin yra tarpas tarp elemento rėmelio ir aplinkinių elementų.
Margin'ai gali persidengti jeigu vienas po kitu einantys elementai yra blokiniai.

Margin iš viršaus ir apačios neveikia linijiniams elementams.

Blokinį elementą, turintį pločio nustatymą, iš kraštų galima sucentruoti naudojant - margin:0 auto;

Margin gali būti nustatomas 8 būdais.

  • margin-top: px;
  • margin-right: px;
  • margin-bottom: px;
  • margin-left: px;
  • margin: top right bottom left;
  • margin: top right&left bottom;
  • margin: top&bottom right&left;
  • margin: all;

Box Sizing

Suteikiant elementui padding ir border, jo turinio dydis didėja. Norint kad elementas išliktų nustatyto pločio ir aukščio, o suteiktas padding ir border įsiskaičiuotų į pradinį dydį, reikia elementui suteikti stilių box-sizing su border-box nustatymu.

Position

  • static (default)
  • relative
  • absolute
  • fixed
  • sticky
  • top, right, bottom, left, z-index (ne pozicijos)

Suteikus elementui betkurią poziciją (apart static), jis įgauna galimybę naudotis top, right, bottom, left ir z-index nustatymais.
z-index nusako elemento eilės sluoksnį ekrane (kaip arti/toli nuo vartotojo elementas yra /// kuris elementas kurį užstos).

Relative

Relative nustatymas suteikia elementui galimybę naudotis top, right, bottom, left ir z-index nustatymais.
Top, right, bottom ir left nustatymai pastumia elementą nuo nurodytos jo pradinės pozicijos per nurodytą atstumą nuo nurodytos kraštinės.
Elemento viduje esant absoliučiajam elementui, jo atitraukimai skaičiuojami nuo artimiausio protėvinio elemento turinčio relatyviąją poziciją.

Fixed

Elementas turintis fixed nustatymą išeina iš puslapio tėkmės.
Fixed elementas prilimpa prie ekrano (viewport) ir juda scroll'inant per svetainę.

Selectors

  • selector1, selector2, ... - keletos selektorių parinkimas
    • tagName.class
    • tagName.class.class2
    • tagName.class#id
    • tagName#id
    • #id.class
    • #id.class.class2
    • .class.class2
  • selector1 selector2 - selector1 viduje betkur (nesvarbu kaip giliai) esantis selector2
  • selector1 > selector2 - selector1 viduje esantis vaikinis selector2
  • selector1 + selector2 - iš karto už selector1 einantis selector2
  • selector1 ~ selector2 - už selector1 einantys selector2
