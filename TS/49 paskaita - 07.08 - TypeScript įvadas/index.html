TypeScript

Typescript yra Javascript kalbos superset'as

TypeScript'as papildo Javascript kalbą tipais. Leidžia programuotojui nustatinėti kintamųjų, parametrų, funkcijos grąžinamų reikšmių, klasių... tipus. (pvz.: Pasakyti, kad kintamasis privalo būti string tipo.)

TypeScript yra labai greitai išpopuliarėjas kalbos superset'as, dėl jo naudos dirbant komandoje arba esant dideliam/tobulėjančiam projektui.

TypeScript tipų nurodymas naudingas, nes jis gaudo įvairias logines klaidas kodo rašymo metu, vietoj to, kad klaidos būtų atvaizduojamos kodo vykdymo metu.

Puslapis nesupranta TS tipo failų, dėl to visviena prikabiname JS tipo failus prie HTML'o. Tam, kad galėtume naudotis TypeScript'u reikia jį kompiliuoti/konvertuoti į JavaScript'ą, ką padarome naudodami tsc komandą, kuria galime naudotis tik tuomet jeigu esame įsirašę typescript'ą į savo įrenginį/projektą.

TypeScript naudojimo įvadas

  1. Įsirašyti TypeScript'ą į įrenginį/projektą.
  2. Susikurti ir parašyti kodą TypeScript faile.
  3. Susikurti ir nusistatyti typescript config failą. (tsc --init)
  4. Sukompiliuoti TS į JS. (tsc --watch)
