Chat GPT

Chat GTP 4.0 yra dalinai nemokamas įrankis suteikiantis limituotą atsakymų kiekį dienoje.
Jis jau yra išbandytas daugelio programuotojų, gyvuojantis ilgą laiką, tai galimai niekur nepradings ir bus toliau tobulinamas/naujinamas ir naudojamas.

Tam, kad pasinaudoti šiuo įrankiu, reikia nueiti į https://chatgpt.com/auth/login puslapį, prisijungti (galima su google paskyra) ir kai būsite nukeltas į pokalbio puslapį/lauką ten galima užduoti klausimus ir sulaukti atsakymų.

GitHub Copilot

GitHub Copilot gali būti naudojamas kaip papildinys (extension) kokiame nors kodo editoriuje arba kaip įrankis GitHub puslapyje.

Jeigu jis yra naudojamas kaip papildinys VisualStudioCode, jis atlieka įvairius dalykus:

  • Generuoja kodą/duomenis (pagal užklausą)
  • Siūlo kodo užbaigimus
  • Klaidų/kodo paiešką (rašant paprastą tekstą)
  • Chat lauką/skiltį
  • ...

Video for more info: Video

Norint naudotis GitHub Copilot reikia turėti VisualStudioCode ir GitHub paskyrą. Nuėjus į šitą nuorodą: https://education.github.com/discount_requests/application ir užpildžius formą galima gauti nemokamą GitHub Copilot licenciją. Formoje reikės pateikti: vardą, pavardę, el. paštą, mokymosi įstaigos pavadinimą, mokymosi įstaigos el. paštą, dokumentą nurodantį, kad esi studentas.
Kol forma yra aplikuojama, galima gauti laikiną nemokamą GitHub Copilot licenziją užpildant formą, kurioje nurodai: gyvenamosios vietos informaciją, kortelės informaciją (mokėjimui, kurį galima bus atšaukti prieš pasibaigiant nemokamai licencijai) ir savo kontaktinę informaciją.
