Komentavimas

  • // - vienos eilutės komentavimas nuo simbolių pradžios
  • /* tarpas */ - daugelio eilučių komentavimas simbolių tarpe

Kodą reikėtų komentuoti. Kad kodas būtų suprantamas jums po ilgo laiko arba kitam programuotojui. Reikia pakomentuoti funkciją / sąlygą / ciklą / klasę..., kad nereikėtų pačiam nagrinėtis ką jinai daro, o pasiskaičius trumpą komentarą - būtų aišku.

JS rašymo būdai

  • Internal - JS rašymas HTML failo viduje, script tag'o viduje.
  • External - JS rašomas atskirame JS faile, kuris yra prijungiamas prie HTML'o.

Pagrinde naudojamas External būdas. Jo prijungimas aprašomas head tag'o viduje. Kadangi HTML failas skaitomas paeiliui, gaunasi, kad JS perskaitomas prieš HTML body, o būna, kad JS kreipiasi į HTML body esančią informciją ir šitaip gaunamas error. Tam, kad nebūtų eiliškumo klaidų, JS prijungime rašomas defer atributas kuris nukelia to failo skaitymą į pabaigą.

Informacijos atvaizdavimas naršyklėje

  • console.log("pranešimas"); - išvedimas į konsolę
  • console.dir("pranešimas"); - išvedimas į konsolę (bus reikalingas ateityje)
  • console.group("pavadinimas"); - sukuria konsolės grupės pradžią (atidarytą)
  • console.groupCollapsed("pavadinimas"); - sukuria konsolės grupės pradžią (uždarytą)
  • console.groupEnd(); - nurodo konsolės grupės pabaigą
  • window.alert("pranešimas"); - puslapyje atvaizduojamas pranešimas
  • window.prompt("pranešimas"); - puspalyje atvaizduojamas pranešimas su vieta atsakymui, kuris grąžinamas į kodą
  • Ateityje: innerHTML/innerText/append...

Kintamieji

Susigalvotas vardas, kurio viduje yra talpinama kokia tais informacija. Tas vardas (kintamasis) gali būti panaudojamas ateityje norint išgauti tą informaciją.

Kintamojo kūrimas

  1. kintamojo startas
    • var - senasis būdas (hoist'ingo pliusas)
    • let - naujasis būdas gali kisti
    • const - naujasis būdas skirtas nekintantiems kintamiesiems
    • - (nieko nerašant) !!nenaudoti!!
  2. kintamojoVardas - susigalvoji logišką ne per ilgą kintamojo vardą be tarpų
  3. = - lygybės/priskyrimo ženklas
  4. reikšmė
    • number - 6|-5|5.5|6516161981|5*6|...
    • string - "dvigubose"|'viengubose'|'du žodž'|`backtick`|...
    • boolean - true|false
    • ...
  5. ; - loginės eilutės pabaiga

Aritmetika

Atliekami kažkokie veiksmai tarp kintamųjų. Galima atlikti veiksmus tarp skirtingo tipo kintamųjų.

  • x + y - sudėtis
  • x - y - atimtis
  • x * y - daugyba
  • x / y - dalyba
  • x % y - liekana
  • x ** y - laipsnio kėlimas
  • x ** (1/y) - šaknies traukimas

Visi aritmetiniai veiksmai APART sudėties gali būti atliekami tik su skaičiais. (95% tiesos)

  • string + string = string
  • string + number = string
  • boolean + string = string
  • boolean + number = number
  • boolean + boolean = number
  • ...
