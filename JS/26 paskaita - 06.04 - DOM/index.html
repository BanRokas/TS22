DOM - Document Object Model

Node

Node'as - HTML elementas JS'o aplinkoje.

Galima HTML'ą pildyti innerHTML'u - rašant HTML kodą JS'e kaip string'ą ir juo papildyti HTML'o elementą. Tuomet tas string'as HTML'e bus skaitomas kaip HTML kodas ir atvaizduojamas kaip HTML kodas.
Šitas būdas turi daug minusų, pradedant nuo saugumo, pabaigiant implementacija.

Geresnis būdas yra pildyti HTML'ą su JS'u - kurti node'us (HTML elementus), juos prilyginti kintamiesiems ir tuomet tuos node'us append'inti į kitus HTML elementus.
Node'ai yra kuriami, modifikuojami, įterpiami į HTML'ą naudojant Node'ų metodus.

Node'ų metodai

  • document.createElement('tagName')
  • document.createTextNode('tekstas koks tik nori...')
  • kam.append(ką0, ką1, ką2...) - į kokį node'ą (kam) gale įdėsime kokį/ius node'ą/node'us/string'us (ką)
  • kam.appendChild(ką) - į kokį node'ą (kam) gale įdėsime kokį node'ą (ką)
  • kam.prepend(ką) - į kokį node'ą (kam) pradžioje įdėsime kokį/ius node'ą/node'us/string'us (ką)
  • kam.setAttribute('name','value')
  • kam.classList
    • add('vienKlase', 'kitaKlase', 'treciaKlase', ...)
    • remove('vienKlase', 'kitaKlase', 'treciaKlase', ...)
    • replace('klaseIsimi', 'klasePridedi')
    • toggle('klase') - jeigu elementas klase turi - ją išima, jeigu neturi - ją prideda
  • ...
