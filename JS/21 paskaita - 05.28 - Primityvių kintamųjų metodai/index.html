Metodai

Metodas - jau sukurta funkcija, kuri priklauso kažkokiam specifiniam elementui/kintamąjam.
Savybė/Property - jau sukurta ir priskirta bei nekintanti reikšmė priklausanti kažkokiam specifiniam elementui/kintamąjam.

Metodai skirti tam, kad pagreitintų/palengvintų programuotojo darbą.

Metodai gali būti prototipiniai (instance) arba statiniai (static).
Prototipinio metodo atveju kreipiamės į metodą ar savybę per kintamąjį.
Statinio metodo atveju kreipiamės į metodą ar savybę per globalų kintamąjį.

Tipas.prototype.metodas/savybė - kintamasis.metodas()/savybe
Tipas.metodas/savybė - Tipas.metodas()/savybe

Boolean

  • prototype.toString()
  • prototype.valueOf()

Number

  • Savybės
    • .EPSILON - grąžina epsilon
    • .MAX_SAFE_INTEGER - skaičius, su kuriuo dar galima daryti veiksmus
    • .MAX_VALUE
    • .POSITIVE_INFINITE
    • .MIN_SAFE_INTEGER - skaičius, su kuriuo dar galima daryti veiksmus
    • .MIN_VALUE
    • .NEGATIVE_INFINITE
    • .NaN - grąžina NaN
  • Statiniai Metodai
    • .isFinite(skaicius) - ar skaičius nėra begalinis
    • .isInteger(skaicius) - ar skaičius yra sveikasis
    • .isSafeInteger(skaicius)
    • .isNaN(kazkas)
    • .parseInt(kazkas) - paverčia į sveikąjį skaičių
    • .parseFloat(kazkas) - paverčia į realųjį skaičių
  • Instance (prototipiniai) Metodai
    • .prototype.toPrecision(nr)
    • .prototype.toFixed(nr) - kiek skaičių už kablelio

Math

  • Savybės
    • .PI - grąžina PI (3.141592653589793)
    • ...
  • Static (ne prototipiniai / statiniai) Metodai
    • .cbrt(nr) - kūbinė šaknis
    • .sqrt(nr) - kvadratinė šaknis
    • .pow(nr, laipsnis) - laipsnio kėlimas
    • .floor(nr) - apvalina į žemesnį
    • .ceil(nr) - apvalina į aukštesnį
    • .round(nr) - apvalina pagal apvalinimo taisykles
    • .hypot(nr1, nr2, nr3, ...) - pitagoras
    • .min(nr1, nr2, nr3, ...) - grąžina mažiausią
    • .max(nr1, nr2, nr3, ...) - grąžina didžiausią
    • .random() - grąžina skaičių tarp 0 ir 1
    • .sign(nr) - tikrina ar skaičius teigiamas(1) ar neigiamas(-1) ar nulis(0)
    • .trunc(nr) - paverčia skaičių sveikuoju / panaikina viską už kablelio
    • ...

String

  • Savybė
    • .length - grąžina string'o ilgį (simbolių kiekį)
  • Instance (prototipiniai) Metodai
    • .prototype.at(nr) - nurodo string'o nr pozicijoje esantį simbolį (galima skaičiuoti nuo galo)
    • .prototype.charAt(nr) - nurodo string'o nr pozicijoje esantį simbolį
    • .prototype.charCodeAt(nr) - grąžina nurodytos raidės UTF-16 bibliotekoje kodą
    • .prototype.concat(string1, string2, string3...) - apjungia du arba daugiau string'us į vieną
    • .prototype.startsWith(text) - ar string prasideda su text
    • .prototype.endsWith(text) - ar string pasibaigia su text
    • .prototype.includes(text) - ar string'e yra text'as
    • .prototype.indexOf(text) - grąžina indeksą pirmojo rasto nurodyto text'o
    • .prototype.lastIndexOf(text) - grąžina indeksą paskutinio rasto nurodyto text'o
    • .prototype.localeCompare(text) - palygina string'ą su text'u
    • .prototype.match(regex) - ar atitinka bent vieną regex
    • .prototype.matchAll(regex) - ar atitinka visus regex
    • .prototype.padStart(ilgis, text) - prideda text pradžioje kol pasiekiamas ilgis
    • .prototype.padEnd(ilgis, text) - prideda text gale kol pasiekiamas ilgis
    • .prototype.repeat(kiek) - kiek kartų atkartoti string'ą
    • .prototype.replace('ka', 'kuom') - pakeičia pirmąjį rastą 'ka" į 'kuom'
    • .prototype.replaceAll('ka', 'kuom') - pakeičia visus rastus 'ka" į 'kuom'
    • .prototype.search(regex)
    • .prototype.slice(nuo, iki) - atkerpa iš string dalį nuo(imtinai) iki iki(neimtinai)
    • .prototype.substring(nuo, iki) - atkerpa iš string dalį nuo(imtinai) iki iki(neimtinai)
    • .prototype.split(skirtukas) - sukarpo string'ą į gabaliukus (pagal skirtukas) ir grąžina masyvą
    • .prototype.toLowerCase() - paverčia visą string į mažąsias raides
    • .prototype.toUpperCase() - paverčia visą string į didžiąsias raides
    • .prototype.trimStart() - panaikina tarpus stringo pradžioje
    • .prototype.trimEnd() - panaikina tarpus string'o pabaigoje
    • .prototype.trim() - panaikina tarpus ir pradžioje ir pabaigoje
