Kintamieji (array ir object)

Jau mokinomės String, Number ir Boolean tipo kintamuosius. Šiandien mokinsimės likusius du Array(masyvas) ir Object(objektas).

Array

Array - masyvas naudojamas grupuoti kintamuosius.
Sintaksė: [true, 'string', 654].
Elementus masyvo viduje pasiekiame (kreipiamės į juos) nurodydami jų eiliškumo numerį. Masyvo eiliškumas prasideda nuo 0.
Į masyvo elementus kreipiamės nurodydami elemento eiliškumo skaičių laužtiniuose skliausteliuose už masyvo pavadinimo.
Eiliškumas YRA svarbus.
Naudojamas norint __IŠVARDINTI__ kažkokius susijusius dalykus.

Object

Object - objektas naudojamas grupuoti kintamuosius.
Sintaksė: {raktinisZodis:'reiksme', raktinisZodis2:56, betkas:false}.
Elementus objekto viduje pasiekiame (kreipiamės į juos) nurodydami jų raktinį žodį.
Eiliškumas nėra svarbus.
Naudojamas norint kažkokį vieną dalyką __APRAŠYTI__.

Tiek objektas, tiek masyvas yra konteineriniai kintamieji, skirti saugoti daug duomenų.

Gali būti nest'inti masyvai ir objektai. Juos mokinsimės ateityje.

Masyvų metodai (ir property)

Masyvų property

Masyvai turi vieną vienintelį property (savybę) pavadinimu "length". Ši savybė nurodo masyvo ilgį (elementų kiekį jo viduje).

Masyvų metodai

  • pop() - išima/iškerpa iš masyvo paskutinį elementą ir grąžina jo reikšmę (tą iškirptą elementą)
  • push(new) - į masyvo galą įdeda/papildo "new" elementu ir grąžina naują masyvo ilgį
  • shift() - išima/iškerpa iš masyvo pirmąjį elementą ir grąžina jo reikšmę (tą iškirptą elementą)
  • unshift(new) - į masyvo pradžią įdeda/papildo "new" elementu ir grąžina naują masyvo ilgį

Ciklai

Yra naudojami norint atlikti kažkokius (vienodus/panašius) veiksmus daugelį kartų.

Ciklų variacijos

  • for(skaitliukoSukūrimas; sąlygaSuSkaitliuku; skaitliukoKitimas){ ... veiksmai ... }
  • while(sąlyga){ ... veiksmai ... }
  • do{ ... veiksmai ... }while(sąlyga)
  • for(const el of array){ ... veiksmai ... }
  • for(const el in object){ ... veiksmai ... }
  • masyvų iteraciniai metodai
