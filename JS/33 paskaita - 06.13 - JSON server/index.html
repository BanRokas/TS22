JSON server

JSON serveris yra NPM (Node Package Manager) paketas, kurio pagalba galime susikurti lokalų API serverį. Šiame serveryje turėti savo susikurtus duomenis, kurie privalo būti JSON faile, JSON formatu. Pasileidus JSON serverį naudojant tą JSON failą su duomenimis, galėsime į jį (serverį) kreiptis su Fetch užklausomis ir siūsti/gauti/redaguoti/trinti jame esančius duomenis.

Tam, kad galėtume naudotis JSON serveriu reikia:

  1. npm (įsirašo kartu su Node'u)
  2. įsirašyti json-server globaliai
  3. susikurti JSON failą su tinkama duomenų struktūra
  4. pasileisti JSON failą su json-server komanda (json-server data.json)

JSON server Routes

*kiekvienas duomenų laukas (dlName) gauna savo route'us

  • GET dlName - gauna visus dlName esančius duomenis
  • GET dlName/id - gauna dlName esantį specifinį duomenį pagal ID
  • POST dlName - papildo dlName esančius duomenis nauju duomenimi
  • PUT dlName/id - redaguoja dlName esantį duomenį pagal ID
  • PATCH dlName/id - redaguoja dlName esantį duomenį pagal ID
  • DELETE dlName/id - trina iš dlName esantį duomenį pagal ID

Terminal

Terminal commands

  • ls - list files - išvardina failus/aplankalus esančius dabartinėje direktorijoje (mac/linux)
  • dir - directories - išvardina failus/aplankalus esančius dabartinėje direktorijoje (windows)
  • cd - change directory - skirta naviguotis tarp aplankalų
  • clear - išvalo komandines eilutės vaizdą (ar kažkas tokios)
  • extra stuff
    • TAB - užbaigia komandinę eilutę arba pasiūlo galimus užbaigimus
    • CTRL C - atšaukia dabartinę komandą/veiksmą/eilutę...
    • ARROW UP/DOWN - vaikšto per komandinės eilutės istoriją

Terminal Navigation

  • / - projekto/sistemos pradžia
  • ./ - niekur (lieki toje pačioje vietoje)
  • ../ - žingsnis per aplankalus atgal
  • - - grįžti į buvusį aplankalą (iš kurio tiką išėjai (tarsi CTRL Z))
