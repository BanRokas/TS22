Prilyginimai (assignment operators)

Kintamųjų reikšmės kodo eigoje keičiasi. Tam kad jas pakeistume reikia prilyginti naujai reikšmei. Tai galima padaryti įvairiais būdais:

  • kintamasis++ - padidina kintamąjį vienetu
  • kintamasis-- - sumažina kintamąjį vienetu
  • ++kintamasis - padidina kintamąjį vienetu
  • --kintamasis - sumažina kintamąjį vienetu
  • kintamasis=kazkas - prilygina kintamojo reikšmę kažkam
  • kintamasis+=kazkas - didina kintamojo reikšmę kažkuom
  • kintamasis-=kazkas - mažina kintamojo reikšmę kažkuom
  • kintamasis*=kazkas - didina dauginant kintamojo reikšmę kažkuom
  • kintamasis/=kazkas - mažina dalinant kintamojo reikšmę kažkuom
  • kintamasis**=kazkas - didina keliant kintamojo reikšmę kažkuo laipsniu
  • kintamasis**=(1/kazkas) - mažina traukiant kažkelinto laipsnio šaknį iš kintamojo
  • kintamasis%=kazkas - randa ir prilygina liekaną dalinant kintamąjį iš kažko

Palyginimai (comparison operators)

Palyginimai yra naudojami norint patikrinti ar reikšmė yra kažkam lygi/nelygi ar didesnė/mažesnė už kažką.

  • x > y - ar x yra daugiau už y
  • x < y - ar x yra mažiau už y
  • x >= y - ar x yra daugiau ARBA lygus y
  • x <= y - ar x yra mažiau ARBA lygus y
  • x == y - ar x ir y reikšmės yra lygios
  • x === y - ar x ir y reikšmės IR tipai yra lygūs
  • x != y - ar x ir y reikšmės nėra lygios
  • x !== y - ar x ir y reikšmės IR tipai nėra lygūs

Loginiai (logical operators)

  • ! - NOT - apverčia reikšmę
  • || - OR - reikalauja, kad abu palyginimai būtų false, kad grąžintų false, kitais atvejais grąžins true
  • && - AND - reikalauja, kad abu palyginimai būtų true, kad grąžintų true, kitais atvejais grąžins false

Tipo (type operators)

  • typeof kazkas - kokio duomens tipo yra kazkas - grąžinamas atsakymas teksto pavidalu
  • kazkas instanceof kazkasKitas - neaktualu (bent jau kolkas)

Sąlygos

Jeigu sąlygos klausimas grąžina tiesą, sąlygos vykdomojo bloko dalies kodas yra vykdomas.

Variacijos

  • if - pagrindinė sąlyga
  • if else - sąlyga ir kiti atvejai
  • if else_if - sąlyga ir kitaSąlyga
  • if else_if else_if else_if...
  • if else_if... else - pagrindinė sąlyga ir kitosSąlyga ir kitiAtvejai
  • switch
  • ternary
  • loginio && sąlyga

If

if(sąlyga) { ... veiksmai tiesos atveju ... }

If else

if(sąlyga) { ... veiksmai tiesos atveju ... } else { ... veiksmai vykdomi jei prieš tai nebuvo nei vienos tiesos ... }

If else_if else

if(sąlyga){ ... veiksmai sąlyga tiesos atveju ... } else if(sąlyga2){ ... veiksmai sąlyga melo ir sąlyga2 tiesos atveju ... } else { ... veiksmai sąlyga melo ir sąlyga2 melo atveju ... }

Truthy ir Falsy

Truthy

  • true
  • non zero number
  • non empty string
  • object
  • array
  • function

Falsy

  • false
  • zero number (0)
  • emtpy string ('')
  • undefined
  • null
  • NaN
