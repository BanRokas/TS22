Funkcijos

Funkcija yra užvadintas kodas, kuris nėra vykdomas kol į jį nėra kreipiamasi. Galima kreiptis daugiau nei vieną kartą.

Funkcija GALI priimti duomenis. Tuos duomenis naudoja savo viduje atliekant veiksmus.

Funkcija gali atlikti kažkokį veiksmą arba grąžinti kažkokius duomenis ARBA abudu.

Parametras - funkcijos viduje naudojamas kintamasis, kurio reikšmė priklauso nuo paduodamo argumento kreipimosi atveju.
Argumentas - reikšmė, kuri yra paduodama į funkciją kreipimosi atveju.

Syntax

function susigalvotasPavadinimas(){ ... veiksmai ... }
function susigalvotasPavadinimas2(parametras, par, par2...){ ... veiksmai su parametrais ... }

Call

susigalvotasPavadinimas();
susigalvotasPavadinimas2(argumentas, arg, arg2...);
